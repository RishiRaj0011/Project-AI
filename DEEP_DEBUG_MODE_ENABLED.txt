================================================================================
CONFIDENCE THRESHOLD: 0.60 (60%)
================================================================================

**Distance to Confidence Conversion:**
```python
distance = face_recognition.face_distance([target], detected)
confidence = 1.0 - distance

If distance <= 0.40: confidence >= 60% ✅ MATCH
If distance > 0.40: confidence < 60% ❌ NO MATCH
```

**Typical Confidence Scores:**
- Same person, good lighting: 70-85%
- Same person, poor lighting: 60-70%
- Same person, different angle: 55-65%
- Different person: 30-50%
